## 打开文件优雅的写法

    file = open("/tmp/foo.txt")
    try:
        data = file.read()
    finally:
        file.close()

相当

    with open("/tmp/foo.txt") as file:
        data = file.read()

这种写法更优雅，文件对象file失效时自动调用关闭文件的函数
